# Employee Payroll JDBC

## Description
Java JDBC project to perform CRUD operations on Employee Payroll database.

## Technologies
- Java
- JDBC
- MySQL
- Maven

## Database Setup
1. Create database payroll_service
2. Create employee_payroll table
3. Insert sample data

## Features
UC1 - Database connection
UC2 - Retrieve employees
UC3 - Update salary
UC4 - PreparedStatement update
UC5 - Retrieve employees by date
UC6 - Add employee
